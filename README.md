# Nowhere Land

## Description
Nowhere Land is a text-based console single player strategy game includes:
* Day and night cycle
* Random events
* 'Feed your people or they die!' system

Game only in **polish version!**

## Used technology
Technology I used in this project:
* C++ 11
* Standard C++ libraries
